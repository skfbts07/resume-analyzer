# Resume Analyzer

## About the Project
This project analyzes resumes using machine learning and natural language processing (NLP).  
It helps recruiters or HR teams automatically evaluate resumes based on skills, experience, and keywords.  
The goal is to speed up candidate screening and improve hiring efficiency.

## Tools & Technologies
- Python  
- Scikit-learn  
- Pandas  
- NumPy  
- NLTK / SpaCy  

## Features
- Text preprocessing and vectorization  
- Model training and evaluation  
- Resume classification based on skill relevance  
- Keyword extraction and scoring
